# claude/ — контекст для Claude

Эта папка исключена из git (`.gitignore`). Файлы помогают Claude быстро
восстанавливать контекст и работать эффективнее между сессиями.

| Файл | Содержимое |
|---|---|
| `project.md` | Цель проекта, структура папок, прогресс по дням |
| `apis.md` | API-ключи, эндпоинты, особенности каждого сервиса |
| `stack.md` | Стек, паттерны разработки, стиль кода, предпочтения |
| `architecture.md` | Архитектура всех приложений: схемы, эволюция, зависимости, справочник модулей |
