# claude/ — контекст для Claude

Дизайн- и контекст-документы проекта. Файлы помогают Claude (и разработчику)
быстро восстанавливать контекст и работать эффективнее между сессиями:
цель проекта, прогресс по дням, архитектура, стек, паттерны.

Папка **версионируется в git** (раньше была в `.gitignore` — теперь в репозитории).

| Файл | Содержимое |
|---|---|
| `project.md` | Цель проекта, структура папок, прогресс по дням |
| `apis.md` | Эндпоинты и особенности сервисов. **Ключи — только плейсхолдеры**; реальные значения задаются в окружении (`os.environ`), не в репозитории |
| `stack.md` | Стек, порты приложений, паттерны разработки, стиль кода, предпочтения |
| `architecture.md` | Архитектура всех приложений: схемы, эволюция по дням, зависимости, справочник модулей, грабли |

> ⚠️ Секреты (API-ключи) в этих файлах **не хранятся** — только плейсхолдеры.
> Перед коммитом проверяйте, что реальных ключей нет.
