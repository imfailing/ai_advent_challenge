# Week 2 / Day 3 — Токены, стоимость и загрузка файлов контекста

Агент считает токены и стоимость в реальном времени, а также позволяет
загружать внешние файлы, которые автоматически добавляются в контекст
каждого запроса к LLM.

## Возможности

- **Подсчёт токенов** — входные, выходные и суммарные по запросу и сессии
- **Подсчёт стоимости** — за текущий запрос и накопленно по всей сессии
- **Загрузка файлов контекста** — txt, md, py, js, json, csv, yaml, html, sql, pdf
- **Персистентность** — история и файлы хранятся в SQLite, сохраняются между перезапусками

## Что считается

| Разрез | Метрики |
|---|---|
| **Текущий запрос** | входные токены (запрос + история + файлы), выходные (только ответ), итого, стоимость, время |
| **Ответ модели** | выходные токены отдельно — видно, сколько стоит генерация |
| **Сессия (накопленно)** | суммарные вход/выход по всем ходам, суммарная стоимость |

Тариф: DeepSeek deepseek-chat, $0.27/1M входных и $1.10/1M выходных токенов (cache miss).

## Почему входные токены растут с каждым ходом

Каждый новый запрос включает **всю историю диалога и все загруженные файлы**:

```
Ход 1:  вход=62   (системный промпт + 1 сообщение)
Ход 2:  вход=84   (системный промпт + 2 сообщения + ответ хода 1)
+ файл: вход=107  (добавился файл контекста в системный промпт)
```

## Загрузка файлов контекста

Файлы добавляются через боковую панель интерфейса (кнопка «+ Загрузить файл»).
Содержимое инжектируется в системный промпт перед каждым запросом — модель
«видит» файл без явного упоминания в чате. Файлы хранятся в SQLite и
восстанавливаются при перезагрузке страницы.

Поддерживаемые форматы: `.txt .md .py .js .ts .json .csv .yaml .yml .html .xml .sql .pdf`
Лимит размера: 512 KB на файл.

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

```bash
export DEEPSEEK_API_KEY="ваш_ключ"
```

## Запуск

```bash
source venv/bin/activate
python app.py
```

Откройте [http://localhost:5000](http://localhost:5000).

## API-маршруты

| Метод  | Путь               | Описание                                           |
|--------|--------------------|----------------------------------------------------|
| GET    | `/`                | Чат с панелью токенов и файлов                     |
| POST   | `/ask`             | Запрос → ответ + `usage` + `session` в JSON        |
| POST   | `/clear`           | Сбросить историю диалога                           |
| POST   | `/upload`          | Загрузить файл контекста (multipart/form-data)     |
| GET    | `/context`         | Список файлов контекста текущей сессии             |
| DELETE | `/context/<id>`    | Удалить файл контекста по id                       |
| GET    | `/sessions`        | Все сессии с суммарными токенами и стоимостью      |
