# Week 2 / Day 3 — Подсчёт токенов и стоимости

Агент считает токены и стоимость в трёх разрезах и показывает их
в боковой панели интерфейса в реальном времени.

## Что считается

| Разрез | Метрики |
|---|---|
| **Текущий запрос** | входные токены (запрос + вся история), выходные токены (только ответ), итого, стоимость, время |
| **Ответ модели** | выходные токены отдельно — видно, сколько стоит именно генерация |
| **Сессия (накопленно)** | суммарные вход/выход по всем ходам, суммарная стоимость |

Тариф: DeepSeek deepseek-chat, $0.27/1M входных и $1.10/1M выходных токенов (cache miss).

## Почему входные токены растут с каждым ходом

Каждый новый запрос включает **всю историю диалога** — так модель помнит
контекст. Чем длиннее разговор, тем больше входных токенов в каждом
следующем запросе:

```
Ход 1:  вход=62  (системный промпт + 1 сообщение)
Ход 2:  вход=84  (системный промпт + 2 сообщения + ответ хода 1)
...
```

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

```bash
export DEEPSEEK_API_KEY="ваш_ключ"
```

## Запуск

```bash
source venv/bin/activate
python app.py
```

Откройте [http://localhost:5000](http://localhost:5000).
После каждого ответа в правой панели обновятся метрики текущего запроса,
полоса соотношения вход/выход и накопленная стоимость сессии.

## API-маршруты

| Метод | Путь        | Описание                                        |
|-------|-------------|-------------------------------------------------|
| GET   | `/`         | Чат с панелью токенов                           |
| POST  | `/ask`      | Запрос → ответ + `usage` + `session` в JSON     |
| POST  | `/clear`    | Сбросить историю и обнулить счётчики в панели   |
| GET   | `/sessions` | Все сессии с суммарными токенами и стоимостью   |
