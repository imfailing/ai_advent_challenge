# Week 2 / Day 3 — Токены, стоимость, файлы контекста и выбор модели

Чат-агент с боковой панелью управления: выбор модели с отображением её
параметров, подсчёт токенов и стоимости в реальном времени, загрузка
внешних файлов в контекст.

## Возможности

- **Выбор модели** — переключение между deepseek-chat и deepseek-reasoner прямо в интерфейсе; отображаются размер контекста, лимит ответа, тарифы и поддержка chain-of-thought
- **Подсчёт токенов** — входные, выходные и суммарные по текущему запросу и всей сессии
- **Подсчёт стоимости** — за запрос и накопленно; стоимость считается по тарифам активной модели
- **Файлы контекста** — загрузка txt/md/py/json/csv/pdf и других форматов; содержимое инжектируется в каждый запрос автоматически
- **Персистентность** — история, файлы и статистика хранятся в SQLite между перезапусками

## Доступные модели

| Модель | ID | Контекст | Макс. ответ | Вход | Выход |
|---|---|---|---|---|---|
| DeepSeek V3 | `deepseek-chat` | 64K | 8K | $0.27/1M | $1.10/1M |
| DeepSeek R1 | `deepseek-reasoner` | 64K | 8K | $0.55/1M | $2.19/1M |

R1 поддерживает chain-of-thought — медленнее и дороже, но точнее на сложных задачах.

## Что считается

| Разрез | Метрики |
|---|---|
| **Текущий запрос** | входные токены (запрос + история + файлы), выходные (только ответ), итого, стоимость, время |
| **Сессия (накопленно)** | суммарные вход/выход по всем ходам, суммарная стоимость |

## Почему входные токены растут с каждым ходом

Каждый запрос включает **всю историю диалога и все загруженные файлы**:

```
Ход 1:  вход=62   (системный промпт + 1 сообщение)
Ход 2:  вход=84   (системный промпт + 2 сообщения + ответ хода 1)
+ файл: вход=107  (файл контекста добавился в системный промпт)
```

## Загрузка файлов контекста

Файлы добавляются через панель «📎 Файлы контекста» (кнопка «+ Загрузить файл»).
Содержимое инжектируется в системный промпт — модель «видит» файл без явного
упоминания в чате. Файлы хранятся в SQLite и восстанавливаются при перезагрузке.

Форматы: `.txt .md .py .js .ts .json .csv .yaml .yml .html .xml .sql .pdf`
Лимит: 5 MB на файл.

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

```bash
export DEEPSEEK_API_KEY="ваш_ключ"
```

## Запуск

```bash
source venv/bin/activate
python app.py
```

Откройте [http://localhost:5000](http://localhost:5000).

## API-маршруты

| Метод  | Путь               | Описание                                              |
|--------|--------------------|-------------------------------------------------------|
| GET    | `/`                | Чат с боковой панелью                                 |
| POST   | `/ask`             | Запрос → ответ + `usage` + `session`                  |
| POST   | `/clear`           | Сбросить историю диалога                              |
| GET    | `/models`          | Список всех моделей с параметрами                     |
| GET    | `/model`           | Активная модель текущей сессии                        |
| POST   | `/model`           | Переключить модель `{"model_id": "..."}`              |
| POST   | `/upload`          | Загрузить файл контекста (multipart/form-data)        |
| GET    | `/context`         | Список файлов контекста текущей сессии                |
| DELETE | `/context/<id>`    | Удалить файл контекста по id                          |
| GET    | `/sessions`        | Все сессии с суммарными токенами и стоимостью         |
