# Day 3 — Четыре способа решения задачи через API

Скрипт решает одну и ту же простую алгебраическую задачу
(«куры и кролики», 35 голов и 94 ноги) четырьмя разными способами:

1. прямой ответ без дополнительных инструкций;
2. с инструкцией «решай пошагово»;
3. сначала модель сама генерирует промпт для решения, затем этот промпт
   используется как инструкция;
4. через группу экспертов (аналитик → инженер → критик).

Сравнение результатов и вывод о том, какой способ оказался наиболее
точным/надёжным — в [results.md](results.md).
Сырой вывод всех запусков — в [output_raw.txt](output_raw.txt).

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

```bash
export DEEPSEEK_API_KEY="ваш_ключ"
```

## Запуск

```bash
source venv/bin/activate
python solve_task.py
```
