# Week 5 / Day 4 — Обязательные источники и цитаты + режим «не знаю»

RAG, где модель **обязана** вернуть структурированный ответ:

- **answer** — сам ответ;
- **sources** — список источников (`source` + `section` + `chunk_id`);
- **quotes** — цитаты (дословные фрагменты из найденных чанков).

Плюс режим **«не знаю»**: если релевантность контекста ниже порога — ассистент
честно отвечает «не знаю» и просит уточнение.

Строится на конвейере day 3 (rewrite + rerank + фильтр по порогу).

---

## Структурированный ответ

Модель вызывается в **JSON-режиме** (`response_format=json_object`) со схемой:

```json
{
  "know": true,
  "answer": "краткий ответ по контексту",
  "sources": [{"source": "файл", "section": "раздел", "chunk_id": "id"}],
  "quotes":  [{"chunk_id": "id", "text": "дословный фрагмент"}],
  "clarification": ""
}
```

Каждый блок контекста помечен `[Источник N | chunk_id=… | файл → раздел]`,
чтобы модель могла сослаться на конкретный чанк.

### Заземление цитат (grounding)

Каждая цитата проверяется: реально ли она есть в найденных чанках —
дословное вхождение **или** ≥85% слов совпадают с одним чанком. Выдуманные
или сильно перефразированные «цитаты» помечаются `grounded=false`.

---

## Режим «не знаю» (усиление)

Двухуровневая защита от выдумывания:

1. **Порог релевантности** (`know_threshold=0.25`): если лучший чанк по
   rerank-скору ниже порога — ассистент сразу возвращает «не знаю» + просит
   уточнение, **не вызывая** LLM для сочинения ответа.
2. **Решение модели**: в промпте — «если контекст не отвечает, ставь
   know=false и задай уточняющий вопрос».

```
❓ Какая погода в Париже?
🤷 НЕ ЗНАЮ (релевантность 0.081 ниже порога)
   Не нашёл в базе релевантной информации. Уточните вопрос…
```

---

## Проверка на 10 вопросах (`evaluate.py`)

| Критерий | Результат |
|---|---|
| Источники есть в ответе | 9/10 |
| Цитаты есть в ответе | 9/10 |
| Цитаты заземлены (из чанков) | 9/10 |
| Смысл ответа ↔ цитаты | 8/10 |
| Режим «не знаю» (3 вопроса вне базы) | **3/3** |

- **Смысл ↔ цитаты** — строгая метрика: ожидаемый факт должен быть и в ответе,
  и подтверждён хотя бы одной **заземлённой** цитатой.
- Все 3 out-of-domain вопроса («столица Австралии», «цена биткоина», «борщ»)
  корректно ушли в «не знаю».
- Небольшая вариативность прогонов: query rewrite недетерминирован, поэтому
  изредка один из 10 вопросов уходит в осторожный know=false.

## Запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
export DEEPSEEK_API_KEY="ваш_ключ"

python build_index.py structural
python app.py "ваш вопрос"    # ответ + источники + цитаты (или «не знаю»)
python evaluate.py            # проверка 10 вопросов + режим «не знаю»
```
