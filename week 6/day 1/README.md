# Week 6 / Day 1 — Локальная LLM (Ollama)

Установка и запуск локальной LLM, обращение к ней через CLI и HTTP API,
3 запроса разной сложности. Всё работает **офлайн, без API-ключей**.

**Инструмент:** [Ollama](https://ollama.com) 0.31.2 · **Модель:** `qwen2.5:1.5b` (986 MB)

---

## Установка и запуск

```bash
# Ollama (если ещё нет): https://ollama.com/download  или  brew install ollama
ollama --version                 # 0.31.2

ollama pull qwen2.5:1.5b         # скачать модель (~1 GB)
ollama list                      # проверить, что модель на месте
```

Ollama поднимает локальный HTTP-сервер на `http://localhost:11434` (стартует
как сервис Ollama.app / `ollama serve`).

---

## Проверки

**1. Модель запускается локально** — сервер отвечает:
```bash
curl -s http://localhost:11434/api/version   # {"version":"0.31.2"}
```

**2. Обращение через CLI:**
```bash
ollama run qwen2.5:1.5b "сколько будет 2+2?"   # → 4
```

**3. Обращение через HTTP API** (нативный `/api/chat`, только stdlib):
```bash
python3 query_local.py           # 3 запроса разной сложности
python3 query_local.py "вопрос"  # свой запрос
```
> В системе может не быть команды `python` — используйте `python3`
> (зависимостей нет, только stdlib, venv не требуется).

Ollama также даёт **OpenAI-совместимый** эндпоинт
`http://localhost:11434/v1/chat/completions` — можно ходить тем же
`openai`-SDK, что и в неделях 1–5 (сменив `base_url`).

---

## 3 запроса разной сложности

| # | Сложность | Запрос | Результат |
|---|---|---|---|
| 1 | простой факт | «Столица Франции?» | ✅ «Париж» (0.2 с) |
| 2 | рассуждение (ловушка) | «3 кота, каждый видит 2 других, сколько всего?» | ⚠️ ответила «4» вместо 3 (1.88 с) |
| 3 | генерация кода | «функция простых чисел до N» | ✅ рабочий код (0.62 с) |

Подробный вывод — в [results.md](results.md).

**Честный вывод:** простой факт и код — корректны; на логической задаче-ловушке
модель 1.5B ошиблась (это ожидаемо для её размера). Для сложного рассуждения
нужна модель побольше (`qwen2.5:7b`, `llama3.1:8b`) — но она медленнее и
тяжелее. Здесь важен факт: локальная LLM **запущена и отвечает** через CLI и API.
