# Тарифы CloudNote

| Возможность | Free | Pro | Business |
|---|---|---|---|
| Устройств | 1 | без ограничений | без ограничений |
| Синхронизация между устройствами | ✗ | ✓ | ✓ |
| Объём хранилища | 1 ГБ | 20 ГБ | 200 ГБ |
| Совместная работа | ✗ | ✓ | ✓ |
| SSO (Google/Microsoft) | ✗ | ✗ | ✓ |
| Админ-панель и роли | ✗ | ✗ | ✓ |
| Приоритетная поддержка | ✗ | ✓ | ✓ (SLA 4 часа) |

## Ключевые ограничения по тарифам
- **Free** — один аккаунт привязан к одному устройству, синхронизация
  недоступна. Чтобы работать с нескольких устройств, нужен Pro или Business.
- **Pro** — синхронизация и совместная работа есть, но **SSO не входит** в тариф.
- **Business** — всё из Pro плюс SSO, админ-панель, SLA.

Сменить тариф: Настройки → Биллинг → Изменить тариф.
